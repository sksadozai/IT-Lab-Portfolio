# Lab 01 - Home Lab Environment Setup

## Overview

This lab focuses on preparing a virtual enterprise environment using Oracle VirtualBox. The objective was to build a complete IT infrastructure that will later be used for Windows Server Administration, Active Directory, Linux Administration, Networking, and Security testing.

---

## Objectives

- Install Oracle VirtualBox
- Create virtual machines
- Allocate system resources
- Prepare an enterprise lab environment

---

## Virtual Machines Created

| Machine | Operating System | Purpose |
|----------|------------------|---------|
| DC01 | Windows Server 2022 Standard | Domain Controller |
| ClientWin11 | Windows 11 Pro | Domain Client |
| UbuntuServer | Ubuntu Server 26.04 LTS | Linux Administration |
| KaliLinux01 | Kali Linux | Security Testing |

---

## Hardware Resources

Host Machine

- RAM: 8 GB
- Storage: 250 GB SSD
- Virtualization: Enabled

---

## VM Configuration

### Windows Server

- 2 GB RAM
- 50 GB Virtual Disk

### Windows 11

- 2 GB RAM
- 50 GB Virtual Disk

### Ubuntu Server

- 1.5 GB RAM
- 25 GB Virtual Disk

### Kali Linux

- 2 GB RAM
- Host-Only + NAT Networking
